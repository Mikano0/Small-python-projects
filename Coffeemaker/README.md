CoffeeMaker (OOP Project)

A console-based coffee maker simulation built using Object-Oriented Programming in Python.  
This project demonstrates classes, objects, and methods to model a coffee machine.

Features
- Make different types of coffee 
- Track ingredients (water, milk, coffee)  
- Refill ingredients and check resource levels  
- Console-based interactive interface
